# Ponydorro

### Pomodorro timer in :unicorn: form

This is a .NET Core 3 WPF [Pomodorro](https://francescocirillo.com/pages/pomodoro-technique) timer.

By default a pomodorro lasts for 25 minutes and a break lasts for 5
